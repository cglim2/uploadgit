먼저 깃허브에서 계정생성 및 레파지토리 생성한달
vs-code 터미널에서 다음과 같이 실행한다

1. git init  => 초기화한다
2. git add .  => 모든 파일을 올린다
3. git status => 상태를 확인한다
4. git commit -m "first commit"
5. git remote add origin https://github.com/cglim2/testgit.git  => 웹의 GitHub.com과 vs-code와 연결
6. git remote -v  =>리모트 확인
7. git push origin master => 깃으로 푸쉬한다
8. git add .  => 소스 수정하고 업로드 진행
9. git status => 원본과 수정본을 확인할 수 있다
10. git commit -m "second commit"  => 두 번재 커밋 메세지
11. git push origin master =>  수정사항 최종 업로드
